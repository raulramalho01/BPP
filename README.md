## Terceira Unidade Trabalho Boas Práticas

Este trabalho tem como objetivo desenvolver a capacidade de diagnosticar e gerenciar dívida técnica, escrever código limpo e reconhecer onde um sistema “deve” qualidade, compreender o custo de conviver com esse débito e decidir sobre o que deve ser refatorado sobre um viés de tempo escasso.
