## Terceira Unidade Trabalho Boas Práticas

Este trabalho tem como objetivo desenvolver, em um cenário simulado, a capacidade de diagnosticar e gerenciar dívida técnica. Mais do que escrever código limpo, o foco é reconhecer onde um sistema “deve” qualidade, compreender o custo de conviver com esse débito e decidir, com critério, o que vale a pena pagar primeiro
